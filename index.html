域名注册 & 部署上线

从购买域名到网站上线的完整阿里云流程。

注册域名

在阿里云万网选购并注册你的域名

1. 访问 wanwang.aliyun.com
2. 搜索你想要的域名
3. 选择 .cn 或 .com 后缀
4. 首年约 ¥29-55 元

ICP 备案

通过阿里云备案系统提交，约 7-14 个工作日

1. 阿里云控制台 → ICP 备案
2. 填写主体信息（个人/企业）
3. 上传身份证正反面
4. 等待管局审核 7-14 天

DNS 解析

添加 CNAME 或 A 记录指向你的服务器

1. 进入域名控制台 → 解析设置
2. 添加 CNAME 记录
3. 记录值填写平台提供的地址
4. TTL 设为 10 分钟即可

上线发布

部署到 Vercel / Cloudflare Pages 等平台

1. GitHub 推送代码
2. Vercel 导入仓库
3. 设置自定义域名
4. 自动 CI/CD 部署 ✓
