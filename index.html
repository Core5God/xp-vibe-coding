从域名注册到全球部署，完整流程指南。📖 查看完整攻略

注册域名

在阿里云轻松选购你的专属标识。

1. 访问 wanwang.aliyun.com
2. 搜索你想要的域名
3. 选择 .cn 或 .com 后缀
4. 首年约 ¥29-55 元

ICP 备案

通过阿里云快速提交。
约 7-14 天即可完成。

1. 阿里云控制台 → ICP 备案
2. 填写主体信息（个人/企业）
3. 上传身份证正反面
4. 等待管局审核 7-14 天

DNS 解析

简单设置 DNS，
将域名指向你的专属空间。

1. 进入域名控制台 → 解析设置
2. 添加 CNAME 记录
3. 记录值填写平台提供的地址
4. TTL 设为 10 分钟即可

上线发布

一键部署至 Vercel 或 Cloudflare，
让世界看见。

1. GitHub 推送代码
2. Vercel 导入仓库
3. 设置自定义域名
4. 自动 CI/CD 部署 ✓
